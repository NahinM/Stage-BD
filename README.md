# Stage-BD
##  Project Description:

StageBD is a web platform designed for Bangladeshi cultural institutions and event organizers to boost real attendance by optimizing the entire event funnel.

- It offers online discoverability, shareable marketing tools, simple ticket reservations, and QR code check-in functionalities.

- Users can search for events by area, date, or genre, reserve seats, receive reminders, and review events.

- Organizers can manage events, seats, and promotions, as well as view analytics to improve outreach.
